# Step to run this project

### How to run locally  

#### step1: create virtual environment
```bash
- conda create -p venv_langchain_all_in_one python=3.11 -y

```

#### step2: Activate virtual environment
```bash
- conda activate venv_langchain_all_in_one 
```
